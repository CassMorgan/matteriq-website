# MatterIQ Website

Marketing website for MatterIQ — AI-first operating system for law firms.

## Pages

- **index.html** — Main landing page (features, pricing, how it works)
- **roi-calculator.html** — Interactive ROI calculator for prospects
- **demo.html** — Demo/walkthrough page
- **one-pager.html** — One-page pitch deck
- **tech-requirements.html** — Technical requirements & specs
- **thirdcoast.html** — ThirdCoast concept page (related project)

## Quick Start

1. Open `index.html` in a browser
2. Navigate to "ROI Calculator" to see the interactive calculator
3. Edit HTML files directly for content updates

## ROI Calculator

The interactive ROI calculator helps prospects understand their specific savings:

**Inputs:**
- Total legal assistants
- Active G&C cases
- Assistant billing rate
- Ward monthly rate
- Unbilled time per day
- Current cases per assistant
- Cases turned away annually
- Infrastructure preference (on-prem vs. managed cloud)
- PC Law integration (yes/no)

**Outputs:**
- Lever 1: Recovered billable hours
- Lever 2: Expanded case capacity
- Lever 3: Avoided annual accounting labor
- Year 1 financial summary
- ROI percentage and payback period
